# EGE Notification Bot

Telegram-бот для автоматического отслеживания результатов ЕГЭ на сайте [РЦОИ Башкортостана](https://rcoi02.ru/gia11_result/).

Бот периодически проверяет сайт, сравнивает данные с локальной базой и отправляет уведомления при появлении или изменении результатов.

## Возможности

- Регистрация через Telegram (ФИО + последние 6 цифр паспорта)
- Проверка данных на сайте РЦОИ перед завершением регистрации
- Фоновый мониторинг результатов для всех зарегистрированных пользователей
- Уведомления о новых баллах и изменениях (статус, апелляция, дата)
- Хранение результатов в SQLite
- Поддержка нескольких пользователей в одном боте

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Регистрация или приветствие |
| `/status` | Текущие результаты из базы |
| `/forcecheck` | Немедленная проверка на сайте |
| `/lastcheck` | Время последней проверки |
| `/stop` | Приостановить уведомления |
| `/resume` | Возобновить уведомления |
| `/cancel` | Отменить регистрацию |

> После первой успешной регистрации уведомления о баллах не отправляются — выполняется только базовая синхронизация. Дальнейшие изменения приходят автоматически.

## Локальный запуск

### 1. Клонировать репозиторий

```bash
git clone https://github.com/nikcto/EGE_notification.git
cd EGE_notification
```

### 2. Создать виртуальное окружение и установить зависимости

```bash
python -m venv .venv
.venv\Scripts\activate        # Windows
pip install -r requirements.txt
```

### 3. Настроить переменные окружения

Создай файл `.env` в корне проекта:

```env
BOT_TOKEN=ваш_токен_от_BotFather
CHECK_INTERVAL=60
TIMEZONE=Asia/Yekaterinburg
```

| Переменная | Обязательно | Описание |
|------------|-------------|----------|
| `BOT_TOKEN` | Да | Токен Telegram-бота |
| `CHECK_INTERVAL` | Нет | Интервал проверки в секундах (по умолчанию `60`) |
| `TIMEZONE` | Нет | Часовой пояс (по умолчанию `Asia/Yekaterinburg`) |
| `TELEGRAM_PROXY` | Нет | Прокси для Telegram, если нужен |

### 4. Запустить бота

```bash
python bot.py
```

## Деплой на Bothost

1. Залей код на GitHub (без `.env`, `.venv`, `data/`, `logs/`)
2. Зарегистрируйся на [bothost.ru](https://bothost.ru/)
3. Создай бота на [bothost.ru/create-bot.php](https://bothost.ru/create-bot.php):
   - **Платформа:** Telegram
   - **Библиотека:** aiogram
   - **Python:** 3.11
   - **Git URL:** ссылка на репозиторий
   - **Ветка:** `main`
   - **Главный файл:** `bot.py`
   - **Использовать собственный Dockerfile:** включить
4. Добавь переменные окружения: `BOT_TOKEN`, `CHECK_INTERVAL`, `TIMEZONE`
5. Останови локальный бот — один токен нельзя использовать в двух местах одновременно

## Технологии

- Python 3.11
- [aiogram](https://docs.aiogram.dev/) 3.x
- SQLite
- requests + BeautifulSoup (парсинг сайта РЦОИ)

## Безопасность

- Не коммить `.env` и токен бота в репозиторий
- Данные для входа на сайт РЦОИ хранятся в локальной SQLite-базе
- Регистрация проходит только при успешной проверке на сайте
